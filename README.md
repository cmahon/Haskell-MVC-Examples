# MVC Examples

Examples for the Haskell MVC library

## Setup

    git clone https://github.com/cmahon/Haskell-MVC-Examples.git
    cd Haskell-MVC-Examples
    cabal sandbox init
    cabal install --only-dependencies
    cabal configure
    cabal build
    cabal run mvc-ex1
    cabal run mvc-ex2
